# Cybersecurity Portfolio

A simple and responsive cybersecurity portfolio website developed using HTML5 and CSS3. This project showcases cybersecurity skills, web security concepts, ethical hacking interests, and malware analysis knowledge through a clean and modern user interface.

## Features

* Responsive landing page
* Modern cybersecurity-themed design
* Navigation bar and hero section
* Skills and services section
* Clean UI using pure HTML and CSS

## Technologies Used

* HTML5
* CSS3

## Purpose

This project was created to practice frontend web development and build a beginner-friendly cybersecurity portfolio website for GitHub and resume projects.
